关于科科机器人

一群怀揣科技梦想的年轻人，用代码与机械谱写智能新篇章

我们是谁

科科机器人科技团队成立于2021年，总部位于深圳，是一家专注于智能机器人研发与自动化解决方案的高科技企业。团队汇聚了来自清华大学、浙江大学、哈工大等顶尖高校，以及大疆、百度、华为等科技企业的优秀人才。

我们坚持"技术驱动、产品为王"的理念，在运动控制、AI视觉、自然语言交互等核心技术上持续深耕，已累计获得23项发明专利与软件著作权。

目前，我们的产品已服务超过36家企业与机构，涵盖教育、工业制造、商业服务等多个领域。

教育系列

KeKe EduBot S1

专为青少年编程教育打造，让每个孩子都能创造属于自己的机器人

EduBot S1 是我们面向K12及高校编程教育推出的旗舰产品。支持 Scratch 图形化编程与 Python 代码编程双模式自由切换，配套200+课时的标准化课程体系。6自由度机械臂扩展模块让学习从虚拟走向现实，内置视觉识别模块支持颜色追踪、标签识别等趣味实验。

编程模式Scratch 3.0 / Python
扩展接口6自由度机械臂 / 传感器套件
视觉模块颜色追踪 / 二维码识别
续航时间4-6小时
适用年龄8岁以上
配套课程200+课时

工业系列

KeKe Arm Pro X3

高精度六轴协作机械臂，重新定义工业自动化

Arm Pro X3 是一款面向精密制造的高性能协作机械臂，采用自研高扭矩无框电机与谐波减速器，重复定位精度达 ±0.02mm。支持力控、阻抗控制、柔顺控制等多种模式，拖拽示教功能让产线部署从未如此简单。

自由度6轴协作
重复精度±0.02mm
负载能力5kg
工作半径850mm
控制模式力控/阻抗/柔顺
部署方式拖拽示教 · 即插即用

服务系列

KeKe Serve Mini

智能商用服务机器人，7×24小时不间断运行的可靠伙伴

Serve Mini 专为酒店、医院、展厅、零售等商用场景设计。搭载自研SLAM导航系统，激光+视觉融合感知，可在复杂动态环境中自主避障导航。基于大语言模型的多轮对话引擎，让机器人真正理解并回应用户的自然语言指令。

导航方式激光SLAM + 视觉融合
交互方式语音 + 触屏 + 手机
续航时间8-12小时
充电方式自动回充
多机协作支持10+台协同
适用场景酒店/医院/展厅/零售
